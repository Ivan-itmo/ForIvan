# Lab3

Apache Ant и Gradle

## Ant

Команды:

```bash
ant compile
ant build
ant test
ant clean
ant music
ant history
```

## Gradle

Команды:

```bash
gradle compile
gradle build
gradle jar
gradle test
gradle clean
gradle music
gradle history
gradle dance
```
